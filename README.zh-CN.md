# No Pain, Same Gain — IGA

[English](README.md) | [Português (BR)](README.pt-BR.md) | [Русский](README.ru.md) | **中文**

**Interference-Gated Aggregation (IGA)** —— 一种具有增长记忆的循环模型，其循环状态在序列长度上恒定，且无困惑度代价，在文献中最难的标准任务上的召回率与全注意力相当。它建立在并改进了基础性的增长记忆表述 GRM / Memory Caching（Behrouz et al., 2026）之上。

## 一句话主张
增长记忆 RNN [Behrouz et al., 2026] 重夺了循环性，但在多查询关联召回（MQAR）上崩溃，且其状态无限增长。IGA 修复了崩溃（将*未归一化*的逐段状态在单一全局比率内门控，以固定记忆作为可证明的初始化下界），通过张量草图多项式路由器按*习得的身份*路由（不读取 token ID），并使状态**在 T 上恒定**（秩 ≤ S 的分解精确移除了 d² 项；封顶记忆限制了增长）—— 在 d=64 时为 0.39 MB，比投射到 1 万亿参数的 KV 缓存小 15×，且在主任务上无召回损失。在官方 Zoology MQAR 测试平台、最难的标准设置 (512,64) 上：IGA 99.2 对 Mamba 23.2 和 Based 79.5，与全 softmax 注意力 100.0 持平。在真实的字符级语言模型（TinyStories，3 个种子，两个上下文）上，IGA 在每个种子的噪声范围内与同等宽度的固定记忆基线在 bits-per-character 上持平 —— **恒定记忆，无困惑度代价**。

## 标题的含义
标题精确命名了测量所支持的结论：*no pain* —— 增长记忆和 O(T²) 步骤被移除，留下恒定状态和每步 O(T) 的计算；*same gain* —— 召回在最难的标准任务上与全注意力持平，困惑度在真实文本上与固定记忆基线持平。

## 诚实的局限性（论文 §8）
关联召回的优势是任务特定的（在 d=128 时不会迁移到自然文本上的 planted-fact 探针）；在别处报告的初步 ~0.016 bpc 增益在 3 个种子、4000 步协议下*不*可复现；扩展 d/ctx 以在真实文本上显现召回差异留作未来工作（受 GPU 预算限制）。召回主张处于小规模（单层，d=64）；语言模型实验是唯一的真实文本设置。
